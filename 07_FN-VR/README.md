## 기능 목록

### 검증보고서

| 기능ID | 기능명 | 설명 | 우선순위 |
|---|---|---|---|
| FN-VR-001 | 검증보고서 목록 조회 | 본인에게 배정된 검증보고서 목록 조회 | 필수 |
| FN-VR-002 | 검증보고서 작성
(심사내용 검증) | 심사보고서에 대한 심사내용 검증 및 검증결과 작성 | 필수 |
| FN-VR-003 | 검증반려 처리 | 보완이 필요한 보고서를 심사팀장에게 반려 | 필수 |
| FN-VR-004 | 검증보고서 열람 | 작성된 검증보고서 내용 열람 | 필수 |
| FN-VR-005 | 검증보고서 수정 | 검증보고서 내용 수정 | 필수 |
